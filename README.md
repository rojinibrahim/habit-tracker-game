# Habit Tracker Game

A Python-based habit tracker with a gamified system.

## Features
- User profiles
- Guild system (Fitness, Health, Education)
- Predefined habits per guild
- Streak tracking
- XP system
- Data persistence using JSON

## How to run
1. Install Python
2. Run:
   python main.py
